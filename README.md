Project Overview
YUSTAZI FASHION Online Clothes Ordering System is a web-based application developed to facilitate online ordering of fashion products. The system enables customers to submit orders through an online form, while the submitted information is stored securely in a MySQL database for management and record keeping.

Problem Statement
Many small fashion businesses rely on manual methods of receiving customer orders, which may lead to loss of information, poor record management, and delays in processing orders.

Objectives
- To provide an online platform for customers to place clothing orders.
- To store customer order information in a database.
- To improve efficiency in managing customer orders.
- To reduce the use of paper-based records.

Technologies Used
- HTML
- CSS
- PHP
- MySQL
- XAMPP

System Functions
The system allows customers to:
- Enter personal information.
- Select clothing products.
- Provide delivery details.
- Submit orders online.
The system stores all order information in a database for future reference and management.

Expected Benefits
- Faster order processing.
- Improved record management.
- Easy access to customer information.
- Better customer service.
